=== Ramazan ===
Tags: ramazan, timings, islam, bangladesh, 


== Description ==

Shows a Calendar of Iftar and sehri Timing on Sidebar of your blog/site. At current it supports 'DHAKA', 'SYLHET', 'CHITTAGONG', 'RAJSHAHI', 'NILFAMARI', 'MOULOVI-BAZAR', 'DINAJPUR', 'CHAA-DANGA' timing. More districts are coming soon.

== Installation ==

1. Download and install the plugin in normal way.
2. Add the widget on your sidebar from wp-admin panel.
